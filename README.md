# ⚡ SelfPower

**SelfPower** is a location-aware hybrid renewable energy optimization platform that helps individuals and small businesses reduce electricity costs and increase energy independence using solar, wind, biogas, and waste-to-energy systems.

---

## 🌍 Problem
Electricity costs are rising globally, and most users cannot decide:
- Should I install solar?
- Is battery worth it?
- Can I use biogas or wind?
- Will I actually save money?

---

## 💡 Solution
SelfPower simulates complete energy systems based on:
- Location (Germany, India, Africa, Finland, or custom)
- User type (home, business, restaurant, farm)
- Energy sources (solar, wind, biogas, waste)
- Battery + grid interaction

It provides:
- Cost savings prediction
- Energy independence score
- System recommendation

---

## ⚙️ System Architecture
- Energy source modules (solar, wind, bio, waste)
- Hourly simulation engine
- Battery storage logic
- Optimization engine
- LLM-based recommendation layer

---

## 📊 Output
- Monthly electricity cost
- Self-sufficiency percentage
- Recommended system setup
- Optional income from energy export

---

## 🚀 Goal
Make renewable energy decisions simple, data-driven, and accessible to everyone.
